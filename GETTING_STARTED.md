## 🚀 准备工作

### 需要的软件

1. **VS Code (代码编辑器)**
   - 下载地址: https://code.visualstudio.com/
   - 点击 "Download" 按钮下载适合你操作系统的版本
   - 安装时保持默认设置即可

2. **Bun (JavaScript 运行环境)**
   - Windows用户:
     ```powershell
     # 在 PowerShell 中运行
     powershell -c "irm bun.sh/install.ps1|iex"
     ```
   - Mac/Linux用户:
     ```bash
     # 在终端中运行
     curl -fsSL https://bun.sh/install | bash
     ```

### 检查安装

打开终端(Windows: PowerShell, Mac: Terminal),输入:

```bash
bun --version
```

如果显示版本号(如 1.2.17),说明安装成功。

## 📦 安装步骤

### 第一步:打开项目

1. 打开 VS Code
2. 点击菜单 `文件` → `打开文件夹`
3. 选择 `m3u8-player` 文件夹
4. 点击 `选择文件夹`

### 第二步:打开终端

在 VS Code 中:
1. 点击菜单 `终端` → `新建终端`
2. 终端会在底部打开

### 第三步:安装依赖

在终端中输入:

```bash
bun install
```

等待安装完成(可能需要几分钟)。

## ▶️ 启动项目

### 启动开发服务器

在终端中输入:

```bash
bun run dev
```

看到以下信息说明启动成功:

```
✓ Ready in 936ms
- Local:        http://localhost:3000
```

### 访问网站

1. 打开浏览器(Chrome, Edge, Firefox, Safari 均可)
2. 在地址栏输入: `http://localhost:3000`
3. 按回车键

现在你应该能看到 M3U8 播放器网站了!

## 🎯 使用功能

### 功能 1: 在线播放 M3U8 视频

1. **找到播放器**
   - 网站首页顶部就是播放器

2. **输入 M3U8 链接**
   - 在输入框中粘贴 M3U8 视频链接
   - 例如: `https://example.com/video.m3u8`

3. **点击播放**
   - 点击蓝色的 "播放" 按钮
   - 视频会自动加载并播放

4. **控制播放**
   - 暂停/播放: 点击视频
   - 进度条: 拖动调整位置
   - 音量: 点击音量图标调整
   - 全屏: 点击右下角全屏按钮

### 功能 2: 下载 M3U8 视频

1. **滚动到下载器部分**
   - 向下滚动页面,找到 "M3U8 下载器" 标题

2. **输入链接**
   - 粘贴 M3U8 视频链接

3. **点击下载**
   - 点击 "Download" 按钮
   - 会显示下载进度(分片数量和百分比)

4. **等待完成**
   - 下载完成后,浏览器会自动保存文件
   - 文件名类似: `video_1234567890.ts`

**下载原理**:
- 程序会自动解析 M3U8 文件
- 下载所有视频分片
- 在你的浏览器中合并
- 保存到下载文件夹

### 功能 3: 转换视频格式

1. **滚动到转换器部分**
   - 找到 "M3U8 转 MP4" 标题

2. **选择输出格式**
   - 点击 MP4, WEBM 或 MKV 按钮
   - 推荐选择 MP4 (最通用)

3. **输入链接并转换**
   - 粘贴 M3U8 链接
   - 点击 "Convert" 按钮

4. **等待处理**
   - 会显示转换进度
   - 完成后自动下载

### 功能 4: 切换语言

1. **找到语言按钮**
   - 在网站右上角,有个地球图标

2. **选择语言**
   - 点击地球图标打开菜单
   - 选择你想要的语言:
     - 简体中文
     - English
     - 繁體中文
     - 日本語
     - Deutsch

3. **界面自动切换**
   - 所有文字会立即切换到选择的语言

## ❓ 常见问题

### Q1: 终端显示"命令未找到"

**问题**: 输入 `bun` 后显示找不到命令

**解决方法**:
1. 重新安装 Bun(参考准备工作部分)
2. 关闭并重新打开终端
3. 如果还不行,重启电脑

### Q2: 安装依赖时出错

**问题**: `bun install` 运行失败

**解决方法**:
1. 检查网络连接
2. 删除 `node_modules` 文件夹(如果存在)
3. 重新运行 `bun install`
4. 如果是网络问题,可以尝试使用VPN

### Q3: 无法访问 localhost:3000

**问题**: 浏览器显示无法连接

**解决方法**:
1. 确认开发服务器正在运行(终端中应该显示 "Ready")
2. 尝试刷新浏览器页面(F5)
3. 尝试其他浏览器
4. 检查是否有其他程序占用 3000 端口

### Q4: 视频无法播放

**可能原因**:
- M3U8 链接无效或已过期
- 视频源有 CORS 限制
- 网络连接问题

**解决方法**:
1. 检查 M3U8 链接是否正确
2. 尝试其他 M3U8 视频链接
3. 查看浏览器控制台的错误信息(F12)

### Q5: 下载速度很慢

**原因**: 取决于视频源服务器速度和网络状况

**优化方法**:
- 项目已使用 5 个并发连接优化速度
- 可以在代码中调整并发数(需要修改代码)
- 等待下载完成,不要中断

### Q6: 转换后的视频无法播放

**可能原因**:
- 转换过程中断
- 源文件有问题

**解决方法**:
1. 重新转换
2. 尝试不同的输出格式
3. 检查源 M3U8 是否可以正常播放

## 🛠️ 修改建议

### 更改网站标题

1. 打开 `src/app/layout.tsx`
2. 找到这一行:
   ```typescript
   title: "M3U8 Player - Free Online M3U8/HLS Player"
   ```

3. 修改引号中的文字
4. 保存文件,刷新浏览器

1. 打开 `src/app/globals.css`
2. 找到 `:root {` 部分
3. 修改颜色值(需要了解 HSL 颜色)
4. 保存并刷新查看效果
